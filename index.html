What is React.js?
React.js is the most popular front-end JavaScript library for building Web applications.
React.js is an open-source JavaScript library that is used for building user interfaces specifically for single-page applications.

Differences between props and state?

Props

  • Come outside;
  • Immutable;
  • Designed to control the component from the outside;

State

  • The internal state of the component;
  • The component itself manages its state (changes it);
  • Designed to save information inside the component during its lifetime;
Hooks

What is the useState Hook?

To make it possible to have the changes reflected in the DOM, we have to use a React hook called useState.

What is the useEffect Hook?

The Effect Hook, just like the name implies, carries out an effect each time there is a state change. By default, it runs after the first render and every time the state is updated.

What is the useCallback Hook?

Returns a memoized callback.
